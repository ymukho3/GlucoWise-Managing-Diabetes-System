# GlucoWise-Managing-Diabetes-System
GlucoWise is a smart mobile app that helps individuals manage diabetes through personalized recommendations and continuous health monitoring.

## 🚀 Features
- 📊 Health tracking (glucose, vitals)
- 🤖 ML-based personalized recommendations
- ⏰ Medication reminders & daily tasks
- 🥗 Lifestyle monitoring
- 🌦️ Context-aware insights
- 💬 Medical chatbot

## 🧠 Approach
Hybrid Recommendation System combining:
- Content-Based Filtering  
- Context-Aware Filtering  

## 🛠️ Tech Stack
- Java (Android Studio)  
- Firebase (Realtime DB, Auth, Storage)  
- Python & PyTorch (ML)
